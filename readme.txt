Overrides the default functionality when clicking on Order notes button so that modal window with order notes is displayed instantly.

== Description ==

Currently, when user navigates to WooCommerce > Orders and clicks on Order Notes icon, it opens the order itself.
WooCommerce Order Notes Window plugin overrides this behaviour by opening modal window with
the content of the Order Notes meta box.

Additionally, information about order note author is stored and displayed.

**WooCommerce 2.0.x is required. The plugin is not compatible with WooCommerce 2.1 or newer.**

== Installation ==

1. Download the latest version and extract it in the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Optionally navigate to WooCommerce Settings > Order Notes Window to adjust default note type.

To disable the feature of order notes window, deactivate the plugin in the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Is it possible to set default type of note which will be automatically selected in the modal window? =

Sure, just navigate to WooCommerce Settings > Order Notes to adjust default note type.
